The code in this repository is a rule based system that applies rules represented in a first order logic system to a knowledge base.

This code implements unify, forward chaining, and backward chaining on pre-defined rules and knowledge base, which can be found in camelot.kb.

## Files:
README.md - this file
camelot.kb - knowledge base
logic_parser.py - parser for first order logic
logic_solver.py - solver with unify, forward chaining, and backward chaining
run_logic.py - script to run the logic solver
test_logic.py - script to run the logic solver tests, feel free to edit and add more tests
forward_chaining_sample_output.txt - sample output of forward chaining
backward_chaining_sample_output.txt - sample output of backward chaining
